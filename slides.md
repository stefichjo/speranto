### esti + stari

| | | | | | |
|-|-|-|-|-|-|
| 🅰 | **esse**  | [ese]       | | **sta**  | [sta]       |
| 🅴 | **essan** | [esan]      | | **stan** | [stan]      |
| 🅾 | **essas** | [esas]      | | **stas** | [stas]      |
| | | | | | |
| 🌍 |           | **[estas]** | |         | **[staras]** |
