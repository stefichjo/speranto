# Lingue

***Lingveo*** estas dialekto de [***Lingvao***](lingua.md). Dialekto de ***Lingveo*** estas [***Lingvoo***](linguo.md). Planlingvoj similaj al ***Lingveo*** estas [***Okcidentalo***](https://eo.wikipedia.org/wiki/Okcidentalo), [***Adjuvilo***](https://eo.wikipedia.org/wiki/Adjuvilo) kaj [***Ido***](https://eo.wikipedia.org/wiki/Ido_(lingvo)).

## Prononco

### Akcento

La antaŭlasta silabo estas akcentata.

| | |
|-|-|
| *bona* | **bona** |

La ŝajnsufiksoj **-ĭn-** kaj **-ĭt** eliziiĝas.

| | |
|-|-|
| *homo* | **homĭne** = **home** |
| *nomo* | **nomĭne** = **nome** |
| *limo* | **limĭte** = **lime** |
| *lumo* | **lumĭne** = **lume** |
| *sango* | **sanguĭne** = **sangue** |

### t, d

**t** estas prononcata *[ŧ]* (= *[θ]*) antaŭ radikofina **-i**. La ŝajnfinaĵo **-ione** *[ione]* mallongiĝas al la ŝajnsufikso **-i** *[j]*.
| | |
|-|-|
| *graco* | **gratie** *[graŧje]* |
| *spaco* | **spatie** *[spaŧje]* |
| | |
| *nacio* | **natie** *[naŧje]* |

Kiam **t** estas prononcata *[t]* antaŭ radikofina **-i**, ĝi portas makronon. (**st** ĉiam prononciĝas *[st]*.)

| | |
|-|-|
| *besto* | **best̄ie** = **bestie** *[bestje]* |

La ŝajnsufikso **-atĭc-** prononciĝas *[ađj]* (= *[aðj]*).

| | |
|-|-|
| *fromaĝo* | **formatĭc̄e** *[formađjo]* |
| *mesaĝo* | **missatĭc̄e** *[misađje]* |
| *vojaĝo* | **viatĭc̄e** *[viađje]* |
 
### c, g

**c** kaj **g** estas prononcataj *[c]* kaj *[dz]* antaŭ **-e** kaj **-i** (kaj **-y**).

| | |
|-|-|
| *centro* | **centre** *[centre]* |
| *leĝo* | **lege**  *[ledze]* |
| | |
| *biciklo* | **bicycle** *[bicikle]* |

Kiam **c** kaj **g** estas prononcataj *[k]* kaj *[g]* antaŭ **e** kaj **i**, ili portas makronon. Kiam ili estas prononcataj *[c]* kaj *[dz]* antaŭ **a**, **o** kaj **u**, ili portas cirkumflekson.

| | |
|-|-|
| *amiko* | **amic̄e** *[amike]* |
| *fungo* | **funḡe** *[funge]* |
| | |
| *paca* | **paĉa** *[paca]* |
| *leĝa* | **leĝa** *[ledza]* |

### i, u

En diftongoj, **i** prononciĝas *[j]*. Komence de radiko, skribiĝas **j** anstataŭ **i**.

| | |
|-|-|
| *majo* | **maie** *[maje]* |
| *familio* | **familie** *[familje]* |
| | |
| *jam* | **jam** *[jam]* |
| *projekto* | **projecte** *[projekte]* |

En diftongoj, **u** prononciĝas *[ŭ]*. Komence de radiko, ĝi skribiĝas **w**.

| | |
|-|-|
| *aŭto* | **aute** *[aŭte]* |
| *akvo* | **aque** *[akŭe]* |
| *kvanto* | **quante** *[kŭante]* |
| | |
| *ŭaŭ* | **wau** *[ŭaŭ]* |

Kiam **i** kaj **u** estas akcentataj en antaŭlasta pozicio, ili portas makronon.

| | |
|-|-|
| *poezio* | **poesīe** *[poezie]* |
| *tatuo* | **tatūe** *[tatue]* |

### v

Postvokala **-b** kaj **-p** ofte fariĝas **-v** *[v]*.

| | |
|-|-|

## Gramatiko

### Substantivoj

Substantivoj finiĝas je **-e**. ***En poezio***, la substantiva finaĵo ellasiĝas.

| | | poezie |
|-|-|-|
| *domo* | **dome** | **domĕ** *[dom]* |

### Adjektivoj

Adjektivoj finiĝas je **-a**. ***En poezio***, ili ne ricevas la multenombran finaĵon **-s**.

| | | poezie |
|-|-|-|
| *bona* | **bona** | |
| *bonaj* | **bonas** | **bona** |

### Adverboj

Adverboj finiĝas je **-o**.

| | |
|-|-|
| *multe* | **multo** |
| *nur* | **solo** |

Celaj adverboj finiĝas je **-u**.

| | |
|-|-|
| *hejme* | **domo** |
| *hejmen* | **domu** |

### Numeraloj

Vd. [numeraloj en ***Lingvao***](lingua.md#numeraloj).

### Artikoloj

La difina artikolo estas **la**. **En poezio**, la multenombra formo de **la** estas **las**.

| | | poezie |
|-|-|-|
| *la altaj domoj* | **la altas domes** | **las altas domes** |

**En poezio**, **la** mallongiĝas al **l'-** antaŭ vokalokomenca vorto respektive al **-l** post vokalofina prepozicio.

| | | poezie |
|-|-|-|
| *de la amo* | **de la amo** | **del amo** |

Ne estas ne-difina artikolo.

### Pronomoj

La personaj pronomoj **olli**, **alli**, **ulli** kaj **elli** estas mallongigeblaj al **li**, kiam la kunteksto estas klara.

| | |
|-|-|
| *li(n)*, *ŝi(n)*, *ri(n)*, *ĝi(n)* | **li** |
| *lia*, *ŝia*, *ria*, *ĝia* | **lia** |

### Verboj

Infinitivoj finiĝas je **-u**.

| | |
|-|-|
| *vidi* | **vidu** |

Verboj finiĝas je **-n** kaj havas tempan vokalismon.

| | |
|-|-|
| *mi vidas* | **mi vidan** |
| *mi vidis* | **mi vidin** |
| *mi vidos* | **mi vidon** |

La volitiva kaj kondicionala vokaloj estas **e** kaj **u**.
 
| | |
|-|-|
| *vidu!* | **viden!** |
| *ke vi vidu* | **c̄e vi viden** |
| *se vi vidus* | **sï vi vidun** |

La radiko de (latin(id)a) verbo estas tiu de la unua persono en ununombro.

| | |
|-|-|
| *mi kondukas* | **mi conducan** |
| *mi legas* | **mi legan** |
| *mi plaĉas* | **mi plaĉan** |
| *mi povas* | **mi possan** |

### Prepozicioj

Prepoziciaĵoj estas poezia formo de substantivo en unu-nombro kun la prepozicio **al**.

| | |
|-|-|
| *ĉirkaŭ* | **circ̄ĕ al** *[cirkal]* |
| *dank' al* | **gratiĕ al** *[graŧjal]* |
| *anstataŭ* | **vicĕ al** *[vical]* |

La akuzativa prepozicio **a** uziĝas ***tra bruo*** (specife antaŭ nomoj aŭ kiam la akuzativa frazparto antaŭas la nominativan frazparton).

| | (tra bruo) |
|-|-|
| *mi vidas Karlan* | **mi vidan a Carla** |
| | |
| *rin mi vidas* | **a ulli mi vidan** |
