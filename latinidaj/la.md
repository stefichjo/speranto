# Lingua Latina

| | | | | |
|-|-|-|-|-|
| **-**   | **-a**  | | **-vi**   | **-veri**  |
| **-ba** | **-re** | | **-vera** | **-visse** |

| | |
|-|-|
| **-m**   | **-i**    |
| **-s**   | **-sti**  |
| **-t**   | **-t**    |
| **-mus** | **-mus**  |
| **-tis** | **-stis** |
| **-unt** | **-unt**  |

* partīunt; capiunt
* legiunt → legunt
* (ā)t → (a)t; (ā)unt → (a)nt:         amāt → amat; amāunt → amant
* amāviunt → amāvērunt
* (iē)t → (ie)t; (iē)unt → (ie)nt:     capiēt → capiet; capiēunt → capient
